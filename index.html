Pizza Heaven

A fun, responsive, and over-the-top styled pizza menu. GitHub Code

  • Custom duration smooth scroll to view total price. Calculated to stop when fully visible at screen bottom.

  • Clicking on a menu item adds it to the receipt and recalculates total price.
